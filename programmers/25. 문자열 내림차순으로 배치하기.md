### 문자열 내림차순으로 배치하기

> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12917



###### 풀이

* 정렬하면 됨

  ```c++
  #include <string>
  #include <vector>
  #include <algorithm>
  
  using namespace std;
  
  string solution(string s) {
      string answer = s;
      
  
      sort(answer.begin(), answer.end(), greater<char>());
      
      return answer;
  
  }
  ```
