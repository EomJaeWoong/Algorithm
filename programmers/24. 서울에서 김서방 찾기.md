# Algorithm

### 서울에서 김서방 찾기

> 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/12919



###### 풀이

* 배열에 있는 "Kim"을 찾으면 됨

  ```c++
  #include <string>
  #include <vector>
  
  using namespace std;
  
  string solution(vector<string> seoul) {
      string answer = "김서방은 ";
      
      for(int i = 0; i < seoul.size(); i++){
          if(seoul[i].compare("Kim") == 0)
              answer += to_string(i);
      }
      answer += "에 있다";
      
      return answer;
  }
  ```
